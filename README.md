# Simple To-Do List Application (PHP + MySQL)

## Thông tin sinh viên
- **Họ và tên:** Phạm Hằng Nga
- **Mã sinh viên:** 2205HTTB035

## Mô tả
Ứng dụng quản lý công việc cá nhân được xây dựng bằng PHP thuần và MySQL.

## Công nghệ sử dụng
- PHP (không sử dụng framework)
- MySQL + PDO
- Bootstrap 5

## Chức năng chính
- Đăng ký, đăng nhập, đăng xuất
- Thêm, sửa, xóa, xem danh sách công việc
- Mỗi người dùng chỉ quản lý công việc của riêng mình

## Cài đặt và chạy
1. Import file `database.sql` vào MySQL
2. Cấu hình kết nối CSDL trong `config.php`
3. Chạy project bằng XAMPP hoặc Laragon
4. Truy cập: `http://localhost/todo_app/login.php`
